## 開発の背景

開発者である山田さんとの出会い・元々学習に力を入れていたReactとの親和性の高さを理由に、
<br />
Yamada UIのキャッチアップを兼ねてミニアプリを作成しました。
<br />
また、「気軽に一言だけ書き残せるようなアプリがあったらいいな」という思いを起点に「一言日記」という形で設計を行いました。
